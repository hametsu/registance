　こんにちは！

　これは `esehara shigeo <http://twitter.com/esehara/>`_ が開発している、ブラウザ上で動くチャットベースのゲーム、「 `レジスタンス・チャット <http://www45045u.sakura.ne.jp/registance/>`_ 」のドキュメントです。始めての人は、このドキュメントを読むことで、「レジスタンス」というゲームの世界へと飛び込むことが可能になるでしょう（それを期待しています！）。

　「レジスタンス・チャット」自体のスクリプトに関して、要望及び改善点がございましたら、「レジスタンス・チャット」用に `Twitterアカウントが存在します <http://twitter.com/resistancechat/>` ので、そちらにリプライを返して頂ければ嬉しいです。
